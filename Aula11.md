# Modelo de caso de Uso - Diagrama
- Composto por duas partes: diagrama e descritivo
- Usado para entender os requisitos Funcionais e os atores que interagem com esse sistema/requisito
- Palavras centrais: Caso de uso
  - o que é? é como eu uso a funcionalidade de um sistema
  - exemplo(funcionalidade): comprar livro
      1) acessar página do site
      2) sistema exibe lista de produtos indicados
      3) usuário pesquisa pelo livro Engenharia de Software, do Pressan
      4) sistema exibe lista de livro de engenharia de software
      5) user seleciona o livro desejado
      6) sistema exibe info. do livro selecionado
      7) user seleciona a opção de comprar agora
      8) sistema solicita login do usuário
      9) user loga no sistema
      10) sistema solicita info. de localização e forma de pagamento
      11) usuário confirma endereço e seleciona PIX
      12) sistema gera cod. PIX e aguarda pagamento
      13) user paga
      14) sistema confirma pagamento e exibe tela de acompanhamento
